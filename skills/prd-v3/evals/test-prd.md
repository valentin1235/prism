# PRD: 수강권 선물하기 기능

## 개요

PODO 사용자가 자신의 수강권(티켓)을 다른 사용자에게 선물할 수 있는 기능을 추가합니다. 이를 통해 사용자 간 바이럴 효과를 극대화하고 신규 유입을 늘립니다.

## 배경

- 기존 사용자가 주변에 PODO를 추천할 때 구체적인 혜택을 제공할 방법이 없음
- 경쟁사(링글, 스픽)는 수강권 공유 기능을 이미 제공 중
- MAU 대비 추천 전환율이 2% 미만으로 낮음

## 기능 요구사항

### FR-1: 수강권 선물 전송
- 사용자는 자신이 보유한 **모든 타입의 수강권**(회차권, 무제한권, 체험권)을 다른 사용자에게 선물할 수 있다
- 선물 시 보유한 수강권에서 지정한 횟수만큼 차감된다
- 무제한권의 경우 **기간을 분할**하여 선물할 수 있다 (예: 30일 중 7일을 선물)

### FR-2: 선물 수강권의 만료 정책
- 선물받은 수강권의 만료일은 **선물 시점으로부터 60일**로 고정한다
- 선물받은 수강권은 원래 수강권의 만료일과 무관하게 독립적으로 운영된다

### FR-3: 선물 쿠폰 자동 발급
- 수강권을 선물받는 사용자에게 **첫 결제 시 30% 할인 쿠폰**을 자동 발급한다
- 이 쿠폰은 **모든 구독 상품**에 적용 가능하다
- 쿠폰 유효기간은 **제한 없음**(무기한)

### FR-4: 구독 해지 시 선물 수강권 처리
- 선물을 보낸 사용자가 구독을 해지하더라도, 이미 선물한 수강권은 **회수하지 않는다**
- 선물받은 사용자가 구독을 해지하면 선물 수강권도 함께 **즉시 만료**된다

### FR-5: 선물 횟수 제한
- 사용자당 월 **최대 10회** 선물 가능
- 같은 사용자에게는 월 **최대 3회**까지 선물 가능

## 비기능 요구사항

### NFR-1: 실시간 알림
- 선물 수신 시 푸시 알림 + 인앱 알림 발송

### NFR-2: 어뷰징 방지
- 동일 IP에서 24시간 내 5회 이상 선물 시 자동 차단

## 성공 지표
- 선물 기능 출시 후 30일 내 MAU 대비 선물 이용률 5% 달성
- 선물 수신자의 유료 전환율 15% 이상
